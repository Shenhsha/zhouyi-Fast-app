# 周易起卦 - 小米手环快应用

基于小米 Vela 快应用框架开发的周易起卦应用，适用于小米手环（胶囊屏设备）。

## 功能特性

- **手动摇卦** — 模拟铜钱投掷，逐爻生成卦象
- **手工指定** — 手动切换各爻阴阳，自由组合卦象
- **时间起卦** — 根据当前时间自动起卦
- **手环自动** — 随机生成卦象
- **断卦规则** — 查看不同变爻数量对应的解卦方法

## 解卦逻辑

根据变爻数量自动匹配对应规则：

| 变爻数 | 规则 |
|--------|------|
| 0 个 | 看本卦卦辞 |
| 1 爻 | 看本卦变爻爻辞 |
| 2 爻 | 看两个变爻爻辞，以上爻为主 |
| 3 爻 | 看本卦卦辞为主，变卦卦辞为辅 |
| 4 爻 | 看变卦两个不变爻爻辞，以下爻为主 |
| 5 爻 | 看变卦中唯一不变的爻辞 |
| 6 爻 | 乾坤用九用六，其他看变卦卦辞 |

## 技术栈

- 框架：小米 Vela 快应用
- 设计宽度：480px
- API Level：2+
- 设备类型：watch

## 安装方式

从 [Releases](https://github.com/Shenhsha/zhouyi-Fast-app/releases) 页面下载最新 `.rpk` 文件，通过小米穿戴应用安装到手环。

## 开发

```bash
# 安装依赖
npm install

# 构装
npm run build

# 构建 RPK 包
npm run release
```

## 许可

本项目包含 64 卦完整数据（卦辞与爻辞），仅供学习和个人使用。
